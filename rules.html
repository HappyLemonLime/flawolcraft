How do I know if something is allowed or not on the server?

The minecraft server follows most of the same rules located in the ⁠rules and info channel on the discord but along with that please refrain from:

1. Don't Discriminate Against Others.

Don't be racist, transphobic or any of the sort. Doing so will result in an imediat ban


2. Respect People's Identities.

3. Do NOT Grief Unprotected Builds.

Even if a build has not been claimed that does not mean that you can break / steal / kill anything that owned by someone.


4. Do not create any NSFW content on the server.

No NSFW Content, such as any builds or map art that may convey any NSFW ideas and or imagery


5. Have common sense!

6. No cheating / hacking. If you are not sure, don't use it or ask a mod!

No use of hacked / cheating clients. Doing so will resualt in a ban from the server. This includes things like x-ray packs.
